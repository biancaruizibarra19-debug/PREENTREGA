Programa Ascenso

Preparación real para las alturas

El programa ascenso está diseñado para llevar tu entrenamiento al siguiente nivel. A través de un plan estructurado de 8 semanas, vas a desarrollar mayor resistencia, fuerza y habilidades técnicas específicas para moverte con confianza en terrenos desafiantes.

Qué incluye el programa

  • Planes para mejorar tu capacidad aeróbica.
  • Desarrollo de habilidades para terrenos variables.
  • Rutinas y hábitos para fortalecer tu enfoque.
  • Fuerza, movilidad y potencia para escalar la montaña.
  • Feedback profesional que te impulsa cada semana.

Requisitos

¿Este programa es para vos?

No todos los perfiles ingresan al Programa ASCENSO. Antes de comenzar, realizamos una entrevista individual para conocer tu experiencia, evaluar tu compromiso y determinar si estás listo para afrontar una preparación técnica de alta exigencia.

Requisitos técnicos

  • Capacidad para caminar largas distancias con carga.
  • Resistencia a entrenamientos de alta exigencia.
  • Tolerancia al frío, fatiga y cambios de ritmo.
  • Experiencia previa en trekking o montaña.
  • Compromiso absoluto con normas de seguridad.

Preguntas frecuentes

¿Qué nivel necesito para entrar a Programa Ascenso?

Necesitás una base física activa y experiencia previa en trekking o montaña. Antes de empezar hacemos una entrevista para evaluar tu perfil.

¿Qué se trabaja durante las 8 semanas?

Entrenamos resistencia, fuerza, técnica de desplazamiento, seguridad y preparación mental para moverte con más criterio en altura.

¿El programa incluye salidas prácticas?

Sí. El plan combina encuentros de entrenamiento con prácticas aplicadas según el avance del grupo y las condiciones de seguridad.

¿Cómo inicio la inscripción a Ascenso?

Completás el formulario, nos contás tu experiencia y coordinamos el primer contacto para confirmar si el programa es adecuado para vos.
